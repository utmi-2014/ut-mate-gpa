===========
ut-mate-gpa
===========

Usage
-----
#. UT mateの単位修得状況照会ページでCtrl-Sでページを保存.(ex. ダウンロード)
#. 同じディレクトリ(ex. ダウンロード)に ``gpa.py`` を置く.
#. ``gpa.py`` を実行::

  python gpa.py
