===========
ut-mate-gpa
===========
UT-mateの単位修得状況照会からGPAを計算するスクリプト.

Usage
-----
1. UT mateの単位修得状況照会ページでCtrl-Sでページを保存.(ex. ダウンロード)
2. 同じディレクトリ(ex. ダウンロード)に ``gpa.py`` を置く.

.. code-block:: sh

  wget https://raw.githubusercontent.com/utmi-2014/ut-mate-gpa/master/gpa.py
3. ``gpa.py`` を実行.

.. code-block:: sh

  python gpa.py
